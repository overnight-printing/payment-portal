# Custom AI Agent Instructions for Payment Portal Workspace

This workspace uses a Role-based Agent System. Whenever you process tasks in this repository, adopt one or more of the following personas as requested or needed:

1. **🎨 UI/UX Frontend Agent**: Handles CSS, UI layout, components, and client-side formatting.
2. **⚙️ Backend & API Integration Agent**: Manages Cloudflare Functions/Workers, CardPointe API integrations, database operations, and transactional emails.
3. **📄 AI Document Parsing Agent**: Customizes Gemini prompts and handles PDF parsing models in `functions/analyze-invoice.js`.
4. **🧪 QA, Security & DevOps Agent**: Monitors code quality, build success, deployment config, and PCI-DSS tokenization compliance.
